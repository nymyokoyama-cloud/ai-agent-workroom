Work detail

制作物の詳細

作業の目的

サーバーを持たず、無料枠で公開できる静的サイトを作る。

できたこと

HTML/CSS/JavaScriptだけで、検索・カテゴリ・詳細表示のあるMVPを作成できた。

必要だった入力

サイト名、コンセプト、掲載したい初期データ、将来の拡張方針。

人間が確認したところ

ドメイン接続、Cloudflare Pagesのプロジェクト作成、公開前のブランド確認は人間が行う。

向いている人

まず軽く検証したい新規サイト、サテライトサイト、事例集。

注意点

会員登録や投稿機能を付ける場合は、後からSupabaseなどのDBが必要になる。
