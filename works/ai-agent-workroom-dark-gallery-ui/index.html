Work detail

制作物の詳細

作業の目的

AIでできた制作物を寝る前にも眺めやすい黒背景の作品ギャラリーとして見せ、詳細ページをSEO資産として増やせる形にする。

できたこと

黒背景、青い発光、画像付き作品カード、詳細ページ、いいね表示、人気順・おすすめ順・将来のピックアップ枠まで実装してCloudflare Pagesへ公開できた。

必要だった入力

サイトの目的、参考画像、既存の静的サイト一式、GitHubとCloudflare Pagesの公開環境。

人間が確認したところ

夜に見ても眩しくない暗さ、作品カードの見せ方、販売導線へつなげる文言は人間が判断するとよい。

向いている人

AIで作った制作物をカード化し、サービス販売や制作依頼につながるポートフォリオにしたい人。

注意点

いいね数は現時点ではローカル保存の試作。全体集計には将来的にDBが必要。
